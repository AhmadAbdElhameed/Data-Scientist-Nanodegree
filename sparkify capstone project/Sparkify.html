Model Selection & Evaluation¶

Metrics¶

  • we have a binary target (churned), so i can use BinaryClassificationEvaluator to evaluate our model.
  • i can set the required column names in rawPredictionCol and labelCol Param and the metric in metricName Param.
  • The default metric for this Evaluator is areaUnderROC.

Accuracy It is the ratio of number of correct predictions to the total number of input samples.It works well only if there are equal number of samples belonging to each class.

Area Under The Curve ROC(Reciever Operating Characteristics) is area under the probability curve. It measures the degree of separability. It tells how much model is capable of distinguishing between classes. Higher the AUC, better the model is at predicting 0s as 0s and 1s as 1s.
